O atributo type="text" está especificando o tipo de valor que aquele input receberá.


Type="button" cria uma botão simples e que é atributo para um evento, no qual o usúario pode clicar. Não é necessário esse input dentro do form.


Type="checkbox" é utilizado para que o usuario escolha mais de uma opção.


Type="date" é utlizado para que o usuário possa escolher uma data


Type="email" é utilizado para que o usuário possa passar uma entrada cujo valor seja um e-mail. Ao usar o atributo required (estudaremos posteriormente sua funcionalidade), o texto que não tenha @ não é reconhecido como type="email."


Type="file" é utilizado para que o usuario possa fazer upload de um arquivo.


Type="number" é utlizado para que o usuario possa passar uma entrada cujo valor seja um numero.


Type="password" é utilizado para que o usuario insira uma entrada cujo o valor seja o de uma senha.Os caracteres insiridos aparecem como ocultos.


Type="radio" são bolinhas em que o usuário irá fazer uma seleção.


Type="range" é uma opção dinâmica, na qual o usuário pode especificar um valor numérico que não deve ser inferior a um determinado valor.


Type="search" é utilizado para que o usuario possa passa uma entrada que permitirá realizar um pesquisa dentro do site. Para realizar essa pesquisa, será feito um filtro, no qual será criada uma lógica no Back-End e integrada ao Front-End.


Type="submit" é utlizado para que o usuario possa enviar uma entrada.


Type="tel" é utilizado para que o usuario possa passar uma entrada cujo valor seja um número
